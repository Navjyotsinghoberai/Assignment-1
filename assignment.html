HTML quotes are bracket that enclose a key word. Every word in the quote has a specific function to perform in HTML

& gt: is a character entity. And is for greater than. When we write > then the result would be for >

& gte: is a character entity. And is for greater than equal to. When we write >e then the result would be for >=

& rdquo: is a character entity. And is for right DOUBLE QUOTE. When we write &rdquo then the result would be for “
